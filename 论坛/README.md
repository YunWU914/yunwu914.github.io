# Mechanical Forum

A static blog/forum application built with React and Vite.

## Features

- Article browsing and searching
- Category filtering
- Like and bookmark functionality
- Responsive design

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build

# Deploy to GitHub Pages
npm run deploy
```

## Tech Stack

- React 18
- Vite 6
- TailwindCSS 3
- React Router DOM
- Lucide React Icons
